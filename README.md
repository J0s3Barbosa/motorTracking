# Node.js Motor Bike Tracking

This an app that offers motor bike services. 
this app uses Node.js, Express, Passport, Mongoose, EJS and some other packages.

### Version: 1.0.0

### Usage

```sh
$ npm install
```

```sh
$ npm start
# Or run with Nodemon
$ npm run dev

# Visit http://localhost:5001


```
